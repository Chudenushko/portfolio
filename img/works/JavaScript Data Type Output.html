    javascript

    Symbols are often used for creating private or internal properties in objects.

javascript
const obj = {
  42: 'forty-two',
  true: 'true',
  [Symbol('symbolKey')]: 'symbol'
};

  • E.F: This selects all elements E that have a class of F. It doesn't involve descendants or children.
